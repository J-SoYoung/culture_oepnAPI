# 🎞 THEATER 🕶<br>
**언제, 어디서나, 편리하게 만나는 생생한 예술의 감동!**<br>
영화를 보는 사람은 많지만 연극과 같이 문화 예술 공연을 보는 사람은 적다.<br>
**연극, 클래식, 전시와 같은 문화를 자주 접해보지 않는 사람들을 위해 <br>
먼저 문화 예술 공연을 경험해 본 사람들이 간단한 정보를 알려주기 위해 만들어진 웹사이트<br>**
